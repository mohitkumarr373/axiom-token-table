\# Axiom Trade Token Table



Pixel-perfect replica of Axiom Trade's token table with Next.js 14, TypeScript, Tailwind CSS.



\## Features



\- Real-time price flash

\- Skeleton loading state

\- Error boundary

\- Responsive to 320px



\## Setup



1\. Clone repo

2\. Install dependencies (`pnpm install` or `npm install`)

3\. Run dev server (`pnpm dev` or `npm run dev`)



\## Deploy



Push to GitHub, and connect to Vercel for instant deployment.
